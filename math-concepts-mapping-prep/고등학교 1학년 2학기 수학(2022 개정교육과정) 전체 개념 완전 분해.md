### **2. 직선의 방정식 (Equation of a Line)**[^9][^2][^3]

#### **2.1 직선의 기울기**[^3]

- 2.1.1 기울기의 정의와 의미
    - 2.1.1.1 기울기(Slope) 정의: 두 점 (x₁, y₁), (x₂, y₂)를 지나는 직선의 기울기 m = (y₂-y₁)/(x₂-x₁)
    - 2.1.1.2 기울기의 기하학적 의미: 직선의 기울어진 정도
    - 2.1.1.3 양의 기울기, 음의 기울기, 기울기 0의 의미
    - 2.1.1.4 기울기가 정의되지 않는 경우: 수직선
- 2.1.2 기울기와 직선의 성질
    - 2.1.2.1 평행선의 조건: 기울기가 같다 (m₁ = m₂)
    - 2.1.2.2 수직선의 조건: 기울기의 곱이 -1 (m₁ × m₂ = -1)
    - 2.1.2.3 기울기를 이용한 직선의 방향 벡터
    - 2.1.2.4 기울기의 부호와 직선의 증감성


#### **2.2 직선의 방정식의 여러 형태**[^9][^3]

- 2.2.1 기울기-절편 형태
    - 2.2.1.1 일반형: y = mx + c (m: 기울기, c: y절편)
    - 2.2.1.2 y절편의 기하학적 의미
    - 2.2.1.3 기울기와 y절편이 주어진 직선의 방정식
    - 2.2.1.4 그래프를 이용한 직선의 방정식 구하기
- 2.2.2 점-기울기 형태
    - 2.2.2.1 한 점 (x₁, y₁)과 기울기 m이 주어진 경우: y - y₁ = m(x - x₁)
    - 2.2.2.2 점-기울기 형태의 활용
    - 2.2.2.3 기울기-절편 형태로의 변환
    - 2.2.2.4 특수한 직선들: 수평선, 수직선
- 2.2.3 두 점을 지나는 직선의 방정식
    - 2.2.3.1 두 점 A(x₁, y₁), B(x₂, y₂)를 지나는 직선
    - 2.2.3.2 기울기를 먼저 구한 후 점-기울기 형태 사용
    - 2.2.3.3 대칭형 방정식: (x-x₁)/(x₂-x₁) = (y-y₁)/(y₂-y₁)
    - 2.2.3.4 특수한 경우 처리: 수직선, 수평선
- 2.2.4 직선의 일반형 방정식
    - 2.2.4.1 일반형: ax + by + c = 0 (a, b는 동시에 0이 아님)
    - 2.2.4.2 일반형에서 기울기-절편 형태로 변환
    - 2.2.4.3 일반형의 장점: 모든 직선을 표현 가능
    - 2.2.4.4 직선의 방정식 간의 상호 변환

### **5. 집합 (Sets)**[^5][^9][^3]

#### **5.1 집합의 기본 개념**[^3]

- 5.1.1 집합의 정의와 표현
    - 5.1.1.1 집합(Set)의 정의: 명확히 구별되는 대상들의 모임
    - 5.1.1.2 원소(Element)와 집합의 관계: ∈, ∉ 기호
    - 5.1.1.3 집합의 표현법
        - 원소나열법: A = {1, 2, 3, 4, 5}
        - 조건제시법: B = {x | x는 자연수, x ≤ 5}
    - 5.1.1.4 유한집합과 무한집합
- 5.1.2 특수한 집합들
    - 5.1.2.1 공집합(Empty Set): ∅ 또는 { }
    - 5.1.2.2 전체집합(Universal Set): U
    - 5.1.2.3 자연수 집합 ℕ, 정수 집합 ℤ, 유리수 집합 ℚ, 실수 집합 ℝ
    - 5.1.2.4 집합의 크기(원소의 개수): |A| 또는 n(A)


#### **5.2 집합 사이의 포함관계**[^9][^3]

- 5.2.1 부분집합의 정의
    - 5.2.1.1 부분집합(Subset): A ⊆ B ⟺ (x ∈ A ⟹ x ∈ B)
    - 5.2.1.2 진부분집합(Proper Subset): A ⊊ B
    - 5.2.1.3 집합의 같음: A = B ⟺ A ⊆ B이고 B ⊆ A
    - 5.2.1.4 공집합은 모든 집합의 부분집합
- 5.2.2 부분집합의 개수
    - 5.2.2.1 n개의 원소를 가진 집합의 부분집합의 개수: 2ⁿ개
    - 5.2.2.2 진부분집합의 개수: 2ⁿ - 1개
    - 5.2.2.3 부분집합의 개수와 조합의 관계
    - 5.2.2.4 특정 조건을 만족하는 부분집합의 개수


#### **5.3 집합의 연산**[^5][^3]

- 5.3.1 기본 연산
    - 5.3.1.1 합집합(Union): A ∪ B = {x | x ∈ A 또는 x ∈ B}
    - 5.3.1.2 교집합(Intersection): A ∩ B = {x | x ∈ A 그리고 x ∈ B}
    - 5.3.1.3 차집합(Difference): A - B = {x | x ∈ A 그리고 x ∉ B}
    - 5.3.1.4 여집합(Complement): Aᶜ = {x | x ∈ U 그리고 x ∉ A}
- 5.3.2 벤 다이어그램 (Venn Diagram)
    - 5.3.2.1 벤 다이어그램의 기본 그리기
    - 5.3.2.2 두 집합의 관계를 벤 다이어그램으로 표현
    - 5.3.2.3 세 집합의 벤 다이어그램

#### **5.4 집합의 연산법칙**[^5]
